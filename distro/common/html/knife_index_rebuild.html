ceth index rebuild¶

The ceth index rebuild subcommand is used to rebuild the search indexes for the open source Seth server. This operation is destructive and may take some time.

Note

This subcommand ONLY works when run against the open source Seth server version 10.x. This subcommand will NOT run against open source seth server 11, Enterprise seth (including hosted Enterprise seth), or Private seth.

Syntax¶

This subcommand has the following syntax:

$ ceth index rebuild

Options¶

Note

Review the list of common options available to this (and all) ceth subcommands and plugins.

This command does not have any specific options.

Examples¶

None.
